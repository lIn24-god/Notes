
## 一、字符串（`s = 'hello world'`）

字符串是字符的序列，在Python中不可变（一旦创建，不能直接修改其中的字符）。

### 1. 字符串连接
使用 `+` 将两个或多个字符串拼接成一个新字符串。  
```python
s1 = 'hello'
s2 = 'world'
result = s1 + ' ' + s2   # 'hello world'
```

### 2. 求字符串长度
`len(s)` 返回字符串中字符的个数（包括空格）。  
```python
len(s)   # 11（因为 'hello world' 有11个字符，空格算1个）
```

### 3. 字符串索引
通过 `[索引]` 访问某个位置的字符。索引从0开始，负数表示从末尾倒着数（-1是最后一个字符）。  
```python
s[0]    # 'h'
s[-1]   # 'd'
```

### 4. 切片操作
`[start:end:step]` 截取子字符串，包含 `start`，不包含 `end`。  
```python
s[0:5]   # 'hello'（索引0~4）
s[6:]    # 'world'（从索引6到末尾）
s[::-1]  # 'dlrow olleh'（反转字符串）
```

### 5. 字符串查找
`find(sub)` 返回子串第一次出现的索引，找不到返回 -1。  
```python
s.find('world')   # 6
s.find('xyz')     # -1
```
另外还有 `index()`（找不到会报错）、`rfind()`（从右查找）等。

### 6. 字符串替换
`replace(old, new)` 将所有 `old` 子串替换为 `new`，返回新字符串（原字符串不变）。  
```python
s.replace('world', 'Python')   # 'hello Python'
```

### 7. 字符串分割
`split(sep)` 按指定分隔符将字符串拆分成列表。不指定分隔符时默认按任意空白字符（空格、换行等）分割。  
```python
s.split()        # ['hello', 'world']
'apple,banana'.split(',')   # ['apple', 'banana']
```

### 8. 字符串格式化
将变量值嵌入到字符串中。常用方法：  
- `f-string`（推荐）：`f"Hello {name}"`  
- `format()`：`"Hello {}".format(name)`  
- `%` 格式化：`"Hello %s" % name`  
```python
name = 'Alice'
f'Hello {name}'   # 'Hello Alice'
```

### 9. 字符串大小写转换
- `upper()`：全部大写  
- `lower()`：全部小写  
- `capitalize()`：首字母大写，其余小写  
- `title()`：每个单词首字母大写  
```python
s.upper()   # 'HELLO WORLD'
s.lower()   # 'hello world'
```

### 10. 字符串去除空白
- `strip()`：去除首尾的空白字符（空格、`\t`、`\n`等）  
- `lstrip()`：去除左侧空白  
- `rstrip()`：去除右侧空白  
```python
'  hello  '.strip()   # 'hello'
```

### 11. 字符串判断
一系列返回布尔值的方法：  
- `isalpha()`：是否全是字母  
- `isdigit()`：是否全是数字  
- `isalnum()`：是否全是字母或数字  
- `isspace()`：是否全是空白字符  
- `startswith(prefix)` / `endswith(suffix)`：是否以某子串开头/结尾  
```python
s.startswith('hello')   # True
'123'.isdigit()         # True
```

## 二、列表和元组

列表（`list`）是可变的有序序列，元组（`tuple`）是不可变的有序序列。  
例子：  
`l = [1,2,3,4,5]`（列表）  
`t = (1,2,3,4,5)`（元组）

以下大部分操作对元组同样适用（除了修改类操作，因为元组不可变）。

### 1. 索引访问列表
使用 `[索引]` 获取元素，索引规则同字符串。  
```python
l[0]    # 1
l[-1]   # 5
```

### 2. 修改列表
通过索引赋值来改变列表中的元素（元组不允许此操作）。  
```python
l[2] = 99   # 列表变为 [1,2,99,4,5]
```

### 3. 求列表长度
`len(l)` 返回列表元素个数。  
```python
len(l)   # 5
```

### 4. 列表切片
与字符串切片完全一致，返回一个新列表。  
```python
l[1:4]   # [2,3,4]
l[::-1]  # [5,4,3,2,1]
```

### 5. 列表追加元素
- `append(x)`：在末尾添加一个元素  
- `extend(iterable)`：将另一个序列中的元素逐个添加到末尾  
- `insert(i, x)`：在索引 i 处插入元素（见下文）  
```python
l.append(6)          # [1,2,3,4,5,6]
l.extend([7,8])      # [1,2,3,4,5,6,7,8]
```

### 6. 列表合并
两种方式：  
- 使用 `+` 运算符（生成新列表）  
- 使用 `extend()`（修改原列表）  
```python
l2 = [6,7]
l + l2          # 新列表 [1,2,3,4,5,6,7]
l.extend(l2)    # 原列表变成 [1,2,3,4,5,6,7]
```

### 7. 列表插入元素
`insert(index, value)` 在指定位置插入元素，原位置及之后的元素右移。  
```python
l.insert(2, 99)   # 在索引2处插入99，列表变为 [1,2,99,3,4,5]
```

### 8. 列表删除元素
多种方式：  
- `pop(index)`：删除并返回指定索引的元素（默认删除最后一个）  
- `remove(value)`：删除第一个匹配的值（找不到会报错）  
- `del l[index]`：删除某个位置或切片  
- `clear()`：清空列表  
```python
l.pop()       # 删除并返回5，列表变为 [1,2,3,4]
l.remove(3)   # 删除第一个3，列表变为 [1,2,4]
del l[0]      # 删除第一个元素，列表变为 [2,4]
```

### 9. 列表排序
- `sort()`：原地排序（修改原列表）  
- `sorted(l)`：返回新排序后的列表，原列表不变  
可选参数 `reverse=True` 降序，`key` 指定排序依据。  
```python
l.sort()             # 升序
l.sort(reverse=True) # 降序
```

### 10. 列表拷贝
- 浅拷贝：`l.copy()` 或 `l[:]` 或 `list(l)`  
- 注意直接赋值（`l2 = l`）只是引用，修改一个会影响另一个。  
```python
l2 = l.copy()   # 独立的副本
```

### 11. 列表循环
使用 `for` 循环遍历列表元素。  
```python
for item in l:
    print(item)
```
如果需要索引，可用 `enumerate()`：  
```python
for i, val in enumerate(l):
    print(i, val)
```

> **关于元组**：元组 `t = (1,2,3,4,5)` 支持索引、切片、长度、循环、合并（生成新元组）、拷贝等**读取**操作，但不支持任何修改（如 `t[0]=10` 会报错）。因为元组不可变，适合作为字典的键或保存不需要改变的数据。

## 三、字典

字典是键值对的无序集合（Python 3.7+ 保留插入顺序），键必须唯一且不可变（常用字符串、数字、元组）。

### 1. 访问字典元素
通过键名获取对应的值，使用 `[key]` 或 `get(key)` 方法（后者在键不存在时返回 None 而不报错）。  
```python
my_dict["name"]       # 'Alice'
my_dict.get("age")    # 25
my_dict.get("gender") # None（不会报错）
```

### 2. 修改字典元素
通过键赋值来修改已有键的值。  
```python
my_dict["age"] = 26   # 年龄变为26
```

### 3. 添加新的键值对
直接给一个不存在的键赋值即可添加。  
```python
my_dict["gender"] = "Female"   # 添加新键'gender'
```

### 4. 删除键值对
- `del dict[key]`：删除指定键  
- `pop(key)`：删除并返回对应的值  
- `popitem()`：删除并返回最后一个插入的键值对（Python 3.7+）  
- `clear()`：清空字典  
```python
del my_dict["city"]
value = my_dict.pop("age")   # 返回25，并删除该键
```

### 5. 求字典大小
`len(my_dict)` 返回键值对的数量。  
```python
len(my_dict)   # 3（初始时）
```

### 6. 字典键列表
`keys()` 返回一个可迭代的视图对象，通常转为列表用。  
```python
list(my_dict.keys())   # ['name', 'age', 'city']
```
同样有 `values()` 和 `items()`（返回键值对元组）。

### 7. 字典遍历
最常用的是遍历 `items()`，同时获取键和值。  
```python
for key, value in my_dict.items():
    print(key, value)
```
也可单独遍历键：`for key in my_dict:` 或 `for key in my_dict.keys():`

### 8. 字典合并
Python 3.9+ 可以使用 `|` 运算符，或使用 `update()` 方法。  
```python
d1 = {"a":1, "b":2}
d2 = {"b":3, "c":4}
d1 | d2          # 生成新字典 {'a':1, 'b':3, 'c':4}
d1.update(d2)    # 原地更新，d1变为 {'a':1, 'b':3, 'c':4}
```

### 9. 字典拷贝
- 浅拷贝：`copy()` 方法或 `dict(my_dict)`  
- 深拷贝：需要 `import copy; copy.deepcopy(my_dict)`（如果字典中包含可变对象）  
```python
new_dict = my_dict.copy()   # 独立的浅拷贝
```
